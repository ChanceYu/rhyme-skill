# 中文押韵规则

收到指令后，按以下顺序处理，**不询问用户，直接输出结果**。

步骤一和步骤二为内部分析步骤，不输出到结果中，直接输出第三步的押韵创作。

## 第一步：识别诗体

按优先级判断：

| 判断条件 | 诗体 |
|----------|------|
| 输入包含词牌名（如"水调歌头"、"西江月"等），或句子字数明显不均匀（含 3、4、6、8 字等混合，不符合标准五言/七言规律） | 词牌体 |
| 每句恰好 7 字（不计标点符号），且内容为诗词 | 七言诗 |
| 每句恰好 5 字（不计标点符号），且内容为诗词 | 五言诗 |
| 其他情况（散文、谚语、俗语、字数不规则等） | 自由体 |

## 第二步：提取韵脚

找出输入内容**各句末字**的韵母：
- 取出现频率最高的韵母作为主韵脚（如 "ang"、"i"、"ou"）
- 若末字韵母各不相同，取**最后一句末字**的韵母作为韵脚基准
- 若输入只有一句，直接取该句末字韵母作为韵脚基准
- 记录韵部名称，以约束生成质量（如：ang 韵、in/en 韵）
- 按现代汉语拼音韵母分类（如 "ang" 来自 "光"、"方"；"in/en" 来自 "心"、"人"）

## 第三步：生成押韵句子

**不询问用户，直接生成 2-3 组押韵句子。**

生成 **2-3 组**押韵句子，规则如下：

**字数规格：**
- 五言体：每句 5 字（不计标点）
- 七言体：每句 7 字（不计标点）
- 词牌体：参照所识别词牌的句式字数（见下方词牌参考表）
- 自由体：每句字数与输入最长句保持一致；若无法确定，默认取 7 字

**押韵要求：**
- 每组末句必须押韵；各组可以使用各自不同的韵脚（不强制与检测韵脚相同），以丰富意境
- 每组 2-4 句；建议组内另有至少 1 句与末句同韵

**风格要求：**
- 各组风格不同，从以下标签中选取不同组合：
  豪放、婉约、清新、沉郁、写景、抒情、禅意、田园
- 不同组之间意象、用词尽量不重复

**输出格式模板（严格遵守）：**

```
**【风格一：{风格标签}】**
{句1}
{句2}
{句3}
{句4}（可选）

**【风格二：{风格标签}】**
{句1}
{句2}
{句3}
{句4}（可选）

**【风格三：{风格标签}】**（若生成第三组）
{句1}
{句2}
{句3}
{句4}（可选）
```

> 以下为常见词牌的句式字数和押韵位置，生成时尽力参考，格律仅供参考。

| 词牌名 | 典型句式 | 押韵位置 |
|--------|----------|----------|
| 水调歌头 | 4/5/5/4/4/6/7/5/5 | 多处，上下片各押韵 |
| 沁园春 | 4/3/4/4/3/4/3/4/4/3 | 上下片分别押韵 |
| 西江月 | 6/6/7/6/6/7 | 上下片分别押韵（仄韵+平韵交替） |
| 满江红 | 4/5/4/7/3/3/4/3/3/7 | 仄韵到底 |
| 菩萨蛮 | 5/5/5/5/5/5/5/5 | 两两换韵 |
| 清平乐 | 4/5/6/6/6/6/6/6 | 上片仄韵，下片平韵 |
| 如梦令 | 6/6/3/6/3/3/6 | 仄韵，第5/6句叠字 |
| 蝶恋花 | 7/7/7/7/7/7/7/7/7/7 | 全篇押韵 |

1. 根据输入句子的实际字数分布推断句式
2. 生成时尽量保持字数一致
3. 在风格标签中注明"词牌格律未知，按输入句式生成"

## 完整输出示例

以下示例展示理想输出格式，作为对齐参考。

### 示例一：七言诗输入

**用户输入：** `/yayun 床前明月光，疑是地上霜`

**识别结果（内部步骤，不输出）：** 七言诗，韵脚：ang（光/霜）

**输出：**

**【风格一：豪放】**
长风破浪会有时，
直挂云帆济沧海，
男儿何不带吴钩，
收取关山五十州。

**【风格二：婉约】**
月落乌啼霜满天，
江枫渔火对愁眠，
姑苏城外寒山寺，
夜半钟声到客船。

**【风格三：清新】**
春风又绿江南岸，
明月何时照我还，
千里共婵娟一梦，
水云深处是家山。

---

### 示例二：谚语/自由体输入

**用户输入：** `/yayun 路遥知马力，日久见人心`

**识别结果（内部步骤，不输出）：** 自由体（5 字句），韵脚：in（心）

**输出：**

**【风格一：沉郁】**
岁月磨人志，
苦难见真情，
风雨同舟处，
方知赤子心。

**【风格二：田园】**
晨露润禾苗，
春耕待秋深，
农家无别事，
守望即初心。

**【风格三：禅意】**
花开不问时，
水流自有音，
万事随缘去，
静处得本心。
