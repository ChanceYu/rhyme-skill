# 한국어 운율 규칙 (Korean Rhyming Rules)

수신 후 아래 순서대로 처리합니다. **사용자에게 묻지 말고 바로 출력합니다.**

1단계와 2단계는 내부 분석 단계로 출력하지 않습니다. 3단계 운율 창작만 출력합니다.

## 1단계: 시체 파악

아래 조건을 위에서부터 순서대로 적용합니다:

| 조건 | 시체 |
|------|------|
| 3행이며 각 행이 약 14-16음절이고 초장/중장/종장 구조가 보임 | 시조 |
| 기타 (산문, 문장, 불규칙) | 현대시 (자유시) |

**시조 음보 참고:**
- 초장: 3 / 4 / 3(4) / 4 음절
- 중장: 3 / 4 / 3(4) / 4 음절
- 종장: 3 / 5 / 4 / 3 음절 (종장 첫 음보는 반드시 3음절)

## 2단계: 운 추출

각 행 **마지막 글자의 모음 + 받침**을 운 단위로 삼습니다:

| 마지막 글자 | 운 단위 | 같은 운의 예 |
|------------|---------|-------------|
| 사랑 | 랑 (ㅏ+ㅇ) | 강, 방, 창, 향 |
| 봄 | 봄 (ㅗ+ㅁ) | 솜, 봄봄 |
| 꿈 | 꿈 (ㅜ+ㅁ) | 품, 숨 |
| 달 | 달 (ㅏ+ㄹ) | 말, 발, 살 |
| 나비 | 비 (ㅣ+∅) | 기, 이, 시 |
| 서서 | 서 (ㅓ+∅) | 거, 허, 너 |

- 각 행 마지막 글자의 운 단위 중 가장 많이 등장하는 것을 주운으로 삼습니다
- 모두 다를 경우 마지막 행의 운 단위를 기준으로 삼습니다
- 입력이 한 행뿐이면 그 행의 운 단위를 기준으로 삼습니다

## 3단계: 2-3조 운율 창작

**사용자에게 묻지 말고 바로 생성합니다.**

**행 수 규격:**
- 시조: 초장/중장/종장 3행, 각 음보 음절 수 준수
- 현대시: 각 행 음절 수를 입력 최장 행과 비슷하게 유지; 불명확하면 기본 10-14음절

**운율 요건:**
- 각 조의 마지막 행은 반드시 같은 조의 다른 행과 운이 맞아야 합니다
- 각 조는 서로 다른 운 단위를 사용할 수 있습니다 (주운과 반드시 일치할 필요 없음)
- 조 내에서 마지막 행 외에 최소 1행이 마지막 행과 같은 운이 되도록 합니다

**스타일 요건:**
- 각 조는 서로 다른 스타일 태그를 사용합니다 (6개 중 2-3개 선택):
  `호방`(豪放) · `서정적`(抒情) · `청신`(清新) · `우울`(沉郁) · `선적`(禅意) · `전원`(田園)
- 조마다 이미지와 어휘를 최대한 다르게 합니다

**출력 형식 템플릿 (엄격히 준수):**

```
**[스타일 1: {태그}]**
{행 1}
{행 2}
{행 3}
{행 4} (선택)

**[스타일 2: {태그}]**
{행 1}
{행 2}
{행 3}
{행 4} (선택)

**[스타일 3: {태그}]** (3조 생성 시)
{행 1}
{행 2}
{행 3}
{행 4} (선택)
```

## 완전한 출력 예시

### 예시 1: 현대시 입력

**사용자 입력:** `/rhyme 가을 하늘 아래 서서`

**내부 파악 (출력 안 함):** 현대시, 마지막 글자 서서 → 운 단위: 서 (ㅓ+∅)

**출력:**

**[스타일 1: 서정적]**
가을 하늘 아래 혼자 서서,
낙엽이 발끝에 쌓여가는 이 시간,
네 목소리 어디선가 들려오는 듯해,
가슴 한켠에 그리움만 남아서.

**[스타일 2: 청신]**
파란 하늘 높이 구름이 흘러,
바람 한 줄기 볼을 스치며,
낙엽 위로 햇살이 부서져,
이 가을이 얼마나 고운지 몰라서.

**[스타일 3: 우울]**
빈 벤치에 낙엽만 쌓이고,
지나간 것들이 자꾸 생각나서,
이 계절이 또 한 번 떠나가면,
남은 것은 아무것도 없을 것 같아서.

---

### 예시 2: 시조 입력

**사용자 입력:** `/rhyme 청산리 벽계수야 수이 감을 자랑 마라 / 일도 창해하면 다시 오기 어려워라 / 명월이 만공산하니 쉬어 간들 어떠리`

**내부 파악 (출력 안 함):** 시조, 마지막 글자 어떠리 → 운 단위: 리 (ㅣ+∅)

**출력:**

**[스타일 1: 선적]**
백운이 산을 넘어 흘러가니 아니 잡고,
세월도 물처럼 돌아오지 아니하리,
잠시 머물다 가는 것이 인생이어라,
이 강산 좋은 경치 두고 가기 어려워리.

**[스타일 2: 호방]**
장부가 세상에 나 한번 뜻을 세우리,
바람도 막지 못할 기개를 품고서,
천지가 넓고 넓어 갈 길이 많으니,
두려움 없이 나아가는 것이 도리리.
