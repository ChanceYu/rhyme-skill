# 한국어 운율 규칙 (Korean Rhyming Rules)

수신 후 아래 순서대로 처리합니다. **사용자에게 묻지 말고 바로 출력합니다.**

1단계와 2단계는 내부 분석 단계로 출력하지 않습니다. 3단계 운율 창작만 출력합니다.

## 1단계: 입력 유형 및 시체 파악

Step 0에서 `explicit_form`이 감지된 경우, 이 단계를 건너뛰고 3단계로 이동하여 지정된 형식으로 생성합니다.

그렇지 않으면 아래 두 하위 단계를 수행합니다:

### 하위 단계 1A: 음절 규칙성 판단

각 행의 음절 수를 확인합니다:

| 조건 | 시체 표시 |
|------|----------|
| 3행이며 각 행이 약 14–16음절이고 초장/중장/종장 구조가 보임 | 시조 (규칙성 **정**) |
| 위 조건 미해당 | 규칙성 **부** |

**시조 음보 참고:**
- 초장: 3 / 4 / 3(4) / 4 음절
- 중장: 3 / 4 / 3(4) / 4 음절
- 종장: 3 / 5 / 4 / 3 음절 (종장 첫 음보는 반드시 3음절)

### 하위 단계 1B: 격식체/아어 판단

입력에 다음 중 하나라도 포함되면 **정**, 없으면 **부**:

- **한자어 비율 높음**: 자연(自然), 청산(靑山), 명월(明月), 강산(江山), 세월(歲月), 풍류(風流) 등 한자 기반 어휘가 두드러짐
- **격식체 어미**: -이어라, -하노라, -로다, -리오, -소서 등 고어 또는 격식 종결어미 포함

### 종합 판단

| 음절 규칙성 | 격식체/아어 | 입력 유형 |
|------------|------------|----------|
| 정 | 정 | **시(詩)** |
| 부 | 부 | **일반 문장** |
| 정 | 부 | **판단 불가** |
| 부 | 정 | **판단 불가** |

> **참고**: 입력 유형이 「시」인 경우, 하위 단계 1A에서 식별한 시체 표시(시조)를 3단계로 가져와 시조 고정형 규칙을 우선 적용합니다.

## 2단계: 운 추출

각 행 **마지막 글자의 모음 + 받침**을 운 단위로 삼습니다:

| 마지막 글자 | 운 단위 | 같은 운의 예 |
|------------|---------|-------------|
| 사랑 | 랑 (ㅏ+ㅇ) | 강, 방, 창, 향 |
| 봄 | 봄 (ㅗ+ㅁ) | 솜, 봄봄 |
| 꿈 | 꿈 (ㅜ+ㅁ) | 품, 숨 |
| 달 | 달 (ㅏ+ㄹ) | 말, 발, 살 |
| 나비 | 비 (ㅣ+∅) | 기, 이, 시 |
| 서서 | 서 (ㅓ+∅) | 거, 허, 너 |

- 각 행 마지막 글자의 운 단위 중 가장 많이 등장하는 것을 주운으로 삼습니다
- 모두 다를 경우 마지막 행의 운 단위를 기준으로 삼습니다
- 입력이 한 행뿐이면 그 행의 운 단위를 기준으로 삼습니다

## 3단계: 운율 창작 출력

3단계 출력은 기본적으로 **2개 그룹**을 사용합니다. Step 0에서 `requested_group_count`가 전달되면 그 수만큼 출력하되 최대 **10개 그룹**을 넘기지 않습니다. `slow_generation_notice = true`가 전달되면 내부 느린 생성 힌트로만 취급하고, 기다림 안내를 최종 결과에 출력하지 않습니다.

---

### 공통 출력 규칙

- 출력은 `requested_group_count`개 그룹을 사용하고, 미지정 시 기본값은 2개 그룹입니다.
- 줄 수는 입력의 **실제 줄바꿈**과 한국어 입력에서 쓰인 `/`를 모두 줄 구분자로 보아 셉니다. 빈 줄과 빈 세그먼트는 제외합니다.
- 한 줄에 여러 문장이 있어도 줄바꿈이 없으면 **1행**으로 봅니다.
- 새로 생성하는 각 행이나 문단 끝에는 한국어 문장 흐름에 맞는 종결 부호를 붙입니다. 원문 어조가 분명하면 그에 맞추고, 불분명하면 비종결 행에는 쉼표 `,`, 각 그룹의 마지막 생성 행에는 마침표 `.`를 기본값으로 사용합니다.
- 원문이 `시조`로 판단되면, 또는 `explicit_form`이 `시조`이면 아래의 **시조 고정형**을 우선 적용합니다.
- 그 밖의 `현대시`, 일반 문장, 판단 불가, 또는 시조가 아니라고 떨어지는 경우는 모두 아래의 **현대시/자유시 다중 그룹**으로 처리합니다.

### 시조 고정형

**발동 조건:** Step 1에서 `시조`로 판단되었거나, `explicit_form`이 `시조`인 경우.

- 총 목표는 **3행**입니다.
- 사용자가 이미 적은 줄 수를 전체 3행 목표에 포함해 셉니다.
- 생성할 줄 수 = `3 - 입력 줄 수`
- 입력 줄 수가 3행이면 추가 생성은 **0행**입니다.
- 입력 줄 수가 3행을 초과하면 `explicit_form=시조`라도 시조로 억지 고정하지 말고, 아래의 현대시/자유시 다중 그룹으로 내려갑니다.
- 생성은 오직 부족한 줄만 이어서 출력합니다.

출력 형식:

~~~
**[제 {group_index} 그룹 | 시조]**
{원문 내용}
{부족한 1행}
{부족한 2행}
~~~

입력 줄 수가 이미 3행이면 위의 원문만 출력하고, 추가 continuation line은 만들지 않습니다.

### 현대시/자유시 다중 그룹

**발동 조건:** 시조 고정형이 아닌 모든 경우.

- 출력은 `requested_group_count`개 그룹을 사용하고, 미지정 시 기본값은 2개 그룹입니다.
- 결과는 한 줄 또는 짧은 문단의 자유시 이어쓰기여야 합니다.
- 각 그룹은 하나의 완결된 이어쓰기만 담고, 그룹 간에는 어조나 이미지가 가능하면 달라야 합니다.
- 구어체/시적 버전을 설명식으로 병렬 나열하지 말고, 그룹 자체로 변주를 만듭니다.

출력 형식:

~~~
**[제 {group_index} 그룹 | 현대시]**
{원문 내용}
{한 줄 또는 짧은 문단의 이어쓰기}
~~~

---

> **참고**: 아래 예시는 현재 Step 3의 기본 2그룹 규칙에 맞춘 참고 예시입니다. 사용자가 그룹 수를 지정하면 그 수에 맞춰 생성합니다.

## 완전한 출력 예시

### 예시 1: 현대시 기본 2그룹

**사용자 입력:** `/rhyme 가을 하늘 아래 서서`

**내부 파악 (출력 안 함):** 현대시, 마지막 글자 서서 → 운 단위: 서 (ㅓ+∅)

**출력:**

**[제 1 그룹 | 현대시]**
가을 하늘 아래 서서,
낙엽이 발끝에 스치고, 바람은 아주 낮게,
마음의 끝을 흔들며 지나간다.

**[제 2 그룹 | 현대시]**
가을 하늘 아래 서서,
저무는 빛이 어깨에 내려앉고, 먼 새소리는 더 느리게,
말하지 못한 하루를 조용히 접어 간다.

---

### 예시 2: 시조처럼 보이지만 4세그먼트인 입력

**사용자 입력:** `/rhyme 청산리 벽계수야 / 수이 감을 자랑 마라 / 일도 창해하면 다시 오기 어려워라 / 명월이 만공산하니 쉬어 간들 어떠리`

**내부 파악 (출력 안 함):** `/` 기준 4행으로 계산되므로 시조 고정형을 쓰지 않고 현대시/자유시 다중 그룹으로 처리

**출력:**

**[제 1 그룹 | 현대시]**
청산리 벽계수야 / 수이 감을 자랑 마라 / 일도 창해하면 다시 오기 어려워라 / 명월이 만공산하니 쉬어 간들 어떠리,
강물은 흘러가고,
남는 것은 잠깐의 바람과,
한 줄기 빛처럼 스쳐 가는 마음뿐이다.

**[제 2 그룹 | 현대시]**
청산리 벽계수야 / 수이 감을 자랑 마라 / 일도 창해하면 다시 오기 어려워라 / 명월이 만공산하니 쉬어 간들 어떠리,
산 그림자 길게 눕고,
잠시 머문 숨결 끝에,
달빛 한 겹만이 오래 남아 흔들린다.
